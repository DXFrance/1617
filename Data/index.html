Et Boontadata, c'est quoi?


Vous souhaitez comparer des moteurs de traitement pour le « Hot Path » ?

Les projets open source boontadata (Github Boontadata) vous permettent de le faire, que ce soit pour des moteurs open source ou Paas Azure.

Les derniers logs d’exécutions sur un scénario commun sont disponibles dans doc/sample_execution_log.txt dans chacun des deux projets.

Le code est disponible dans code comme par exemple code/spark/master/code/StreamingJob.scala dans le premier projet ou code/paas-artifacts/StreamAnalytics/query.sql dans le second.
